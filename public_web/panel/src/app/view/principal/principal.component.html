Servidores

Los servidores operan a través de una arquitectura cliente-servidor. Los servidores son programas de computadora en ejecución que atienden las peticiones de otros programas.

Conexiones

La conexión es el enlace que se establece entre el emisor y el receptor a través del que se envía el mensaje. Comúnmente puede referirse al tipo de enlace

API REST

Es una interfaz de programación de aplicaciones (Application Programming Interface). Es un conjunto de rutinas que provee acceso a funciones de un software.

Seguridad
